Bem-vindo ao Simulador de Provas

Use as opções abaixo para começar.

Tutorial rápido

  1. Criar uma prova: acesse a página Administração e clique em "Criar prova".
  2. Executar uma prova: na lista de provas, clique em "Aplicar prova".
  3. Importar via JSON: vá para Importar JSON e envie um arquivo.
  4. Importar via PDF: vá para Importar PDF e selecione o documento.
